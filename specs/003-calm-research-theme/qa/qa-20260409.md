### Test Case QA-TC-001: Custom CSS Rule Integration

**Expected**: Adding a custom CSS rule to the theme does not cause console errors and CSS is included in theme instance

**Test Method**: `test_custom_css_integration_basic`

**Test Implementation**:
```python
def test_custom_css_integration_basic(self):
    custom_css = """
    :root {
        --primary-accent: #3b82f6;
    }
    .research-input {
        background-color: var(--primary-accent);
    }
    """
    theme = CalmResearchTheme(custom_css=custom_css)
    assert theme.custom_css == custom_css
    assert len(theme.custom_css) > 0
    assert "--primary-accent" in theme.custom_css
```

**Execution Result**: ✅ PASSED

**Evidence**: Test assertion passed, no exceptions raised
